# Hackathon Raia 2025

Projeto do Hackathon Raia 2025.

## Tecnologias

- Node.js
- Express (API)
- Axios (API)
- React
- Python
- ChatGpt 4-o mini

## Instruções de uso

### Para o frontend (`front`):

```bash
npm install
npm run dev
```

### Para a API (`api rest`):

```bash
npm install
npm install express
npm install axios
```
### Para a conexão com a IA (`chatbot`):
```bash
pip install -r requirements.txt
python openai_chat.py
```
